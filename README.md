# AshDiebackGenomics
This repository contains the data, pipelines and scripts that are needed to reproduce the results of this article:

Szukala, A., Unger, G.M., Trujillo-Moya C., Geburek, T., Kirisits, T., Schueler, S., and Konrad, H. (2025). Limitations of *ad hoc* genotyping in detecting ash dieback tolerance in *Fraxinus excelsior*. *Tree genetics & Genomes*, [doi: 10.1007/s11295-025-01725-6](https://doi.org/10.1007/s11295-025-01725-6)
