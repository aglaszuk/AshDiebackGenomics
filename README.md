# AshDiebackGenomics
This repository contains the data, pipelines and scripts that are needed to reproduce the results of the following preprint:

Szukala, A., Unger, G.M., Trujillo-Moya C., Geburek, T., Kirisits, T., Schueler, S., and Konrad, H. (2024). Limitations of *ad hoc* genotyping in detecting ash dieback tolerance in *Fraxinus excelsior*.
