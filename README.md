# AshDiebackGenomics
This repository contains the data, pipelines and scripts that are needed to reproduce the results of the following publication:

Szukala, A., Unger, G.M., Trujillo-Moya C., Geburek, T., Kirisits, T., Schueler, S., and Konrad, H. (2024). Identifying disease-tolerant genotypes of Fraxinus excelsior using ad hoc genotyping tools is challenging: A case study on ash dieback.
